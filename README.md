# Mobile Game

## Overview
This mobile game is built using Python and leverages Kivy for the interface, Pygame for animations and sound effects, and SQLite for storing high scores. The game features touch detection, animation control, speed variation, a score system, sound effects, and a user interface with a start/restart button.

## Features
- Touch Detection
- Animation Control
- Speed Variation
- Score System
- Sound Effects
- User Interface
- Settings Menu
- Leaderboard
- Pause/Resume
- Tutorial

## Installation
1. Ensure Python is installed on your system.
2. Install the required dependencies using pip:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Game
Navigate to the `/app` directory and run the `main.py` file:
```bash
python main.py
```

## File Structure
- `/app`: Contains the main application files.
- `/assets`: Contains image and sound assets.
- `/utils`: Contains utility functions and database interactions.
- `/tests`: Contains unit tests for the game logic and UI components.
